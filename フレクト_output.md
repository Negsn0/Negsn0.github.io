
# 株式会社フレクト 最終面接対策シート

---

## 🎯 面接全体の目的
最終面接では「スキル」よりも「姿勢」と「価値観の一致」を重視。  
特に以下の4要素を一貫して伝えることが重要。

| 軸 | 面接官が見たいポイント | あなたが出すべき要素 |
|----|--------------------------|------------------------|
| 自律性 | 自分の意思で考え行動できるか | 研究の方向性・工夫・行動を“自分発”で語る |
| 知的好奇心 | 学びを楽しみ、掘り下げているか | Julia導入や理論探求への姿勢 |
| チーム性 | 協働の中で価値を発揮できるか | 共同研究や議論での役割 |
| キャリア意識 | 将来像が具体的か | 技術を軸に価値提供・影響力を持ちたい意志 |

---

## 🧭 面接の流れ想定
1. 冒頭：代表から「逆質問どうぞ」と促される（1〜2問）  
2. 自己紹介・研究説明（短く）  
3. 掘り下げ質問（研究・学び方・好奇心の方向性）  
4. キャリア観・フレクトとの接点  
5. 最後に印象・質問など

---

## 🧩 2. 自己紹介（40秒）
> 千葉大学大学院で原子核構造の理論研究をしています。  
> 原子核の形や超伝導性を数値的に解析し、独自にコードを構築して効率化を行ってきました。  
> 試行錯誤を重ねながら、複雑な現象を構造化して整理する過程に面白さを感じ、  
> 今では「論理的に課題を解決する」こと自体が自分の軸になっています。

## 🧠 3. 想定質問と回答戦略

| 質問                  | 面接官の狙い     | 回答戦略                                                      |
| ------------------- | ---------- | --------------------------------------------------------- |
| 研究で一番大変だったことは？      | 主体性・問題解決力  | 「自分で定義した課題→試行錯誤→成果」の流れを明確に。例：収束しない計算を安定化するアルゴリズムを自作。      |
| 周囲を巻き込んだ経験は？        | 協働性・発信力    | 研究室でJuliaを導入・共有した話が最適。<br>「他の学生にも便利だと感じて共有→輪が広がった」という流れで。 |
| なぜJuliaを選んだのか？      | 知的好奇心と目的意識 | 「言語自体より、研究の本質に集中できる仕組みを作りたかった」<br>→“技術を手段として使う姿勢”を見せる。    |
| 研究を通して得た一番の学びは？     | 自律性・思考力    | 「誰も答えを教えてくれない中で、自分の手で構造を理解する面白さ」<br>→“自分で考える習慣”を強調。       |
| チームで自分はどんな立ち位置？     | チーム適応性     | 「冷静に全体を見て、理論的に整理し方向を示すタイプ」<br>→リーダーではなく、“まとめ役・理論担当”でも良い。  |
| 6年後どんなエンジニアになっていたい？ | キャリア意識     | 「技術の枠にとどまらず、課題発見から提案までできる人」<br>→“影響力を持つ人間になりたい”で締める。      |

## 💡 4. フレクトとのマッチングを語る例（志望動機強化） 
> フレクトの「あるべき未来 をクラウドでカタチにする」という言葉には、  
> 技術を目的ではなく“社会の仕組みをより良くするための手段”として扱う考え方が表れています。  
> 私自身も、研究を通じて「仕組みを理解し、再現性をもって改善する」ことに喜びを感じており、  
> その姿勢がフレクトの文化と非常に近いと感じています。  
> 将来的には、技術面で価値を出しながら、チームや後輩に良い影響を与えられるような人材を目指します。

## 🚀 5. 面接官が惹かれる話し方のポイント
- 「◯◯したからこそ△△がわかった」と**原因→気づき**で語る  
- 「私は〜タイプの人間だと思います」と**自己分析を自分の言葉で**  
- 「今後はそれを〜に活かしたい」と**未来志向で締める**  
- 自慢ではなく、「面白かった・学びがあった」という**素直なトーン**
